사용 방법

이 HTML 파일은 독립적으로 작동하는 한국 문화 갤러리 웹사이트입니다. 아래와 같은 기능을 제공합니다:

  • 반응형 디자인: 데스크톱과 모바일 모두 최적화
  • 인터랙티브 카드: 호버 시 3D 변환 효과
  • 터치 지원: 모바일에서 터치 인터랙션 지원
  • 성능 최적화: 이미지 지연 로딩(lazy loading)

GitHub에 업로드하기: 이 파일을 index.html로 저장한 후 GitHub 저장소에 업로드하세요.

# GitHub에 업로드하는 방법
1. GitHub 저장소 생성
2. index.html 파일 업로드
3. GitHub Pages 설정에서 소스 브랜치 선택
4. https://[사용자명].github.io/[저장소명] 으로 접속
